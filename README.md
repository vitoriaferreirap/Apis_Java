# Api_Java

## HttpServer.java

### Servidor HTTP funcional, desenvolvido manualmente, que:

- Abre uma porta TCP (ServerSocket)
- Escuta conexões de rede
- Aceita um cliente (Socket)
- Recebe bytes (InputStream)
- Interpreta esses bytes como texto HTTP
- Escreve bytes de resposta HTTP (OutputStream)
- Fecha a conexão
